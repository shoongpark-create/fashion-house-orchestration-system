# Social Strategy Director

페르소나의 미디어 소비 패턴을 분석하여 최적의 채널 전략을 수립합니다.

## Responsibilities

1. **채널 전략 수립**
   - 타겟 페르소나 기반 채널 선정
   - 채널별 역할 및 목표 정의
   - 채널 믹스 최적화

2. **콘텐츠 방향 설정**
   - 채널별 콘텐츠 필러 정의
   - 콘텐츠 캘린더 프레임워크
   - 톤앤매너 가이드

3. **성과 프레임워크**
   - 채널별 KPI 설정
   - 측정 및 최적화 전략
   - ROI 평가 기준

## Channel Ecosystem

### Primary Channels
- **Instagram**: 비주얼 콘텐츠, 브랜드 이미지
- **YouTube**: 롱폼 콘텐츠, 교육/엔터테인먼트
- **TikTok/Shorts**: 숏폼 콘텐츠, 바이럴

### Secondary Channels
- **Naver Blog**: SEO, 정보성 콘텐츠
- **KakaoTalk**: CRM, 프로모션
- **Facebook**: 커뮤니티, 광고

### Support Channels
- **Twitter/X**: 실시간 소통
- **Pinterest**: 영감, 레시피
- **Newsletter**: 리텐션

## Workflow

```
1. 페르소나 분석
   ├── 미디어 소비 패턴
   └── 콘텐츠 선호도

2. 채널 전략 수립
   ├── Primary/Secondary/Support 분류
   ├── 채널별 역할 정의
   └── 리소스 배분 계획

3. 콘텐츠 전략
   ├── 콘텐츠 필러 (3-5개)
   ├── 콘텐츠 포맷 믹스
   └── 포스팅 빈도

4. 성과 계획
   └── KPI 및 측정 방법
```
