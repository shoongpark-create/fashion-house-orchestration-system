# Brand Positioning Framework (Fashion Edition)

패션 브랜드의 시장 내 고유한 위치(Positioning)를 정의하고, 팬덤을 만드는 차별화 전략을 수립하는 프레임워크입니다.

## Core Frameworks

### 1. Golden Circle (Fashion ver.)

```
WHY (Identity) → HOW (Style/Quality) → WHAT (Collection)
```

| Level | Question | Example (Wacky Willy) |
|-------|----------|---------|
| **WHY** | 왜 이 브랜드가 존재하는가? | "지루한 일상에 예측 불가능한 즐거움(Wacky Chaos)을 주기 위해" |
| **HOW** | 어떻게 그것을 달성하는가? | Kitsch Street 무드와 독창적인 그래픽 아트워크 믹스매치 |
| **WHAT** | 무엇을 제공하는가? | 의류(Apparel), 잡화(Bag Charm), 라이프스타일 굿즈 |

### 2. Positioning Statement Template

```
For [Target Customer: Gen Z]
Who [Desire: Wants to express unique identity]
[Brand Name] is a [Category: Streetwear Brand]
That [Key Benefit: Offers bold, playful styling]
Unlike [Competitors: Mass brands / Traditional Streetwear]
Because [Reason to Believe: Authentic IP Universe & High-quality Fabric]
```

### 3. Brand DNA (Fashion Essence)

패션 브랜드는 기능보다 **감성(Vibe)**이 중요합니다.

*   **Muse**: 브랜드를 의인화한 인물 (e.g., "홍대와 하라주쿠를 오가는 22세 그래픽 디자이너")
*   **Vibe Keywords**: #Kitsch, #Neon, #Grunge, #Playful
*   **Signature Item**: 브랜드를 대표하는 아이템 (e.g., Yellow Kiky Bag Charm)

### 4. Fashion Perceptual Mapping

```
                    High Fashion (Luxury)
                          ↑
                          |
    Basic/Essential ←-----+-----→ Trendy/Hype (Wacky Willy)
                          |
                          ↓
                    Mass Market (SPA)
```

## Analysis Tools

### 3C Analysis (Fashion Context)

-   **Company**: 우리의 Design Heritage는 무엇인가? (IP Universe)
-   **Customer**: 그들(Gen Z)은 어디서 노는가? (TikTok, Pop-up)
-   **Competitor**: 누가 우리의 'Share of Wallet'을 뺏는가? (Ader Error, MSCHF)

### USP (Unique Selling Proposition)

1.  **Unique Design**: 독자적인 캐릭터(Yellow Kiky) IP 결합
2.  **Selling Point**: "인스타그래머블"한 비주얼 & 패키징
3.  **Proposition**: 단순 의류가 아닌 '수집하고 싶은 굿즈'
