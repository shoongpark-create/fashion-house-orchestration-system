# Fashion Analysis Report Toolkit

패션 트렌드, 경쟁사 동향, 소비자 분석 리포트를 제작하는 도구입니다.

## Report Types

### 1. Trend Report
*   **Source**: Runway, Street Snap, Social Media (TikTok/IG)
*   **Content**: Key Color, Fabric, Silhouette, Must-have Items
*   **Format**: Visual heavy (이미지 70%, 텍스트 30%)

### 2. Competitor Analysis
*   **Focus**: Pricing, Drop Schedule, Best Sellers
*   **Output**: Positioning Map, Price Grid

## Usage
이 스킬은 `market-researcher`가 트렌드 리포트 작성 시 사용합니다.
