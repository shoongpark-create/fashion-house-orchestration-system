# 데이터 분석 패턴

이 문서는 마케팅 분석 리포트에서 자주 사용되는 데이터 분석 패턴과 프레임워크를 제공합니다.

## 시장 규모 분석

### TAM/SAM/SOM 계산

#### Top-Down 접근법

```markdown
TAM = 전체 시장 규모 (글로벌/전체 산업)
SAM = TAM × 접근 가능 비율 (지역, 세그먼트 제한)
SOM = SAM × 현실적 점유율 목표
```

**예시**:
```
TAM: $500B (글로벌 패션 시장)
SAM: $50B (프리미엄 패션, 아시아-태평양)
SOM: $500M (한국 프리미엄 패션, 5년 목표)
```

#### Bottom-Up 접근법

```markdown
SOM = (목표 고객 수) × (평균 거래액) × (구매 빈도)
SAM = SOM / 목표 점유율
TAM = SAM × 시장 확장 배수
```

**예시**:
```
목표 고객: 100,000명
평균 거래액: $50/월
구매 빈도: 12회/년
SOM = 100,000 × $50 × 12 = $60M
```

### CAGR 계산

```markdown
CAGR = ((최종값 / 초기값)^(1/기간) - 1) × 100%
```

**예시**:
```
2024년 시장: $10B
2034년 예상: $25B
기간: 10년

CAGR = ((25/10)^(1/10) - 1) × 100%
     = (2.5^0.1 - 1) × 100%
     = 9.6%
```

## 재무 분석 패턴

### ROI 계산

```markdown
ROI = ((수익 - 비용) / 비용) × 100%
```

### NPV 계산

```markdown
NPV = Σ (현금흐름_t / (1 + r)^t) - 초기투자

r = 할인율
t = 기간 (년)
```

### Payback Period

```markdown
Payback = 초기투자 / 연간 현금흐름
```
