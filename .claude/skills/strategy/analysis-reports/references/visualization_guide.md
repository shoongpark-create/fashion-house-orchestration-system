# 시각자료 생성 가이드

이 문서는 마케팅 분석 리포트에 필요한 시각자료의 프롬프트 템플릿과 생성 가이드를 제공합니다.

## 필수 시각자료 (6개)

### 1. 시장 성장 추이 차트

```
시장 성장 막대 그래프:
- X축: 연도 (2020-2034)
- Y축: 시장 규모 (Billion USD)
- 역사적 데이터: 진한 파랑 (#1a365d)
- 전망 데이터: 연한 파랑 (#63b3ed)
- 각 막대에 수치 레이블 표시
- CAGR 주석 추가
- 깔끔한 비즈니스 스타일
```

**파일명**: `01_market_growth.png`

### 2. TAM/SAM/SOM 다이어그램

```
동심원 TAM SAM SOM 다이어그램:
- 가장 큰 원: TAM (Total Addressable Market) - 연한 파랑
- 중간 원: SAM (Serviceable Addressable Market) - 중간 파랑
- 가장 작은 원: SOM (Serviceable Obtainable Market) - 진한 파랑
- 각 원에 시장명과 금액 표시 ($X Billion)
- 깔끔한 비즈니스 다이어그램 스타일
- 우측에 범례 포함
```

**파일명**: `02_tam_sam_som.png`

### 3. Porter's Five Forces 다이어그램

```
Porter 5 Forces 분석 다이어그램:
- 중앙: 산업 내 경쟁 (Industry Rivalry) - 오렌지
- 상단: 신규 진입자 위협 (Threat of New Entrants) - 파랑
- 하단: 대체재 위협 (Threat of Substitutes) - 파랑
- 좌측: 공급자 교섭력 (Bargaining Power of Suppliers) - 파랑
- 우측: 구매자 교섭력 (Bargaining Power of Buyers) - 파랑
- 각 박스에 HIGH/MEDIUM/LOW 표시
- 화살표로 방향성 표시
- 깔끔한 전략 컨설팅 스타일
```

**파일명**: `03_porters_forces.png`

### 4. 경쟁 포지셔닝 매트릭스

```
2x2 경쟁 포지셔닝 매트릭스:
- X축: 시장 집중도 (Niche ↔ Mass Market)
- Y축: 솔루션 접근방식 (Traditional ↔ Innovative)
- 4개 사분면에 기업 로고/이름 배치
- 각 사분면에 레이블 (Leaders, Challengers, Niche Players, Visionaries)
- 기업 위치는 원형 마커로 표시
- 마커 크기는 시장 점유율 비례
- 깔끔한 컨설팅 스타일
```

**파일명**: `04_competitive_positioning.png`

### 5. 리스크 히트맵

```
리스크 히트맵 매트릭스:
- X축: 영향도 (Impact) - Low, Medium, High
- Y축: 발생 확률 (Probability) - Low, Medium, High
- 색상 그라데이션: 초록(낮음) → 노랑(중간) → 빨강(높음)
- 각 셀에 리스크 ID (R1, R2, ...) 배치
- 우측에 리스크 ID와 설명 범례
- 깔끔한 비즈니스 스타일
```

**파일명**: `05_risk_heatmap.png`

### 6. 실행 타임라인

```
24개월 실행 로드맵 Gantt 차트:
- 4단계 Phase 구분 (Foundation, Growth, Expansion, Optimization)
- 각 Phase에 3-4개 마일스톤
- 월별 타임라인 표시
- Phase별 색상 구분
- 주요 마일스톤에 다이아몬드 마커
- 현재 시점 표시 (점선)
- 프로페셔널 프로젝트 관리 스타일
```

**파일명**: `06_implementation_timeline.png`

## 추가 시각자료 (필요시)

### 7. 세그먼트 분포 파이 차트

```
시장 세그먼트 파이 차트:
- 4-6개 세그먼트로 분할
- 각 조각에 세그먼트명과 비율(%) 표시
- 가장 큰 세그먼트는 약간 분리 (exploded)
- 비즈니스 컬러 팔레트 사용
- 우측에 범례 포함
```

**파일명**: `07_segment_distribution.png`

### 8. 지역별 시장 분포

```
지역별 시장 분포 트리맵:
- 면적 = 시장 규모
- 색상 = 성장률 (진할수록 높은 성장)
- 지역: North America, Europe, Asia-Pacific, Latin America, MEA
- 각 영역에 지역명, 금액, 성장률 표시
```

**파일명**: `08_regional_distribution.png`

### 9. SWOT 분석 매트릭스

```
SWOT 분석 2x2 매트릭스:
- 좌상단: Strengths (강점) - 초록
- 우상단: Weaknesses (약점) - 빨강
- 좌하단: Opportunities (기회) - 파랑
- 우하단: Threats (위협) - 오렌지
- 각 사분면에 3-5개 bullet points
- 깔끔한 전략 컨설팅 스타일
```

**파일명**: `09_swot_analysis.png`

### 10. 기술 로드맵

```
기술 발전 로드맵 타임라인:
- 3개 시기: Current (현재), Near-term (1-3년), Long-term (3-5년)
- 각 시기에 주요 기술 트렌드 나열
- 화살표로 기술 진화 방향 표시
- 아이콘 또는 심볼 활용
```

**파일명**: `10_technology_roadmap.png`

### 11. 고객 여정 맵

```
고객 여정 맵 플로우차트:
- 5단계: Awareness, Consideration, Purchase, Retention, Advocacy
- 각 단계에 터치포인트 표시
- 감정 곡선 오버레이 (만족도)
- Pain points와 Opportunities 주석
```

**파일명**: `11_customer_journey.png`

### 12. 밸류 체인 다이어그램

```
밸류 체인 다이어그램:
- 주요 활동 (Primary Activities): 5개 화살표 연결
- 지원 활동 (Support Activities): 상단에 4개 레이어
- 각 활동에 간단한 설명
- 마진 표시 우측에
```

**파일명**: `12_value_chain.png`

## 프롬프트 작성 가이드라인

### 스타일 키워드

| 목적 | 키워드 |
|------|--------|
| 비즈니스 문서 | `clean business style`, `professional`, `corporate` |
| 컨설팅 리포트 | `McKinsey style`, `consulting diagram`, `strategic` |
| 데이터 시각화 | `data visualization`, `infographic`, `chart` |
| 프레젠테이션 | `presentation ready`, `slide-ready`, `executive summary` |

### 컬러 팔레트

```
Primary:    #1a365d (Navy)
Secondary:  #2b6cb0 (Blue)
Accent:     #ed8936 (Orange)
Success:    #38a169 (Green)
Warning:    #d69e2e (Yellow)
Danger:     #e53e3e (Red)
Background: #f7fafc (Light Gray)
Text:       #2d3748 (Dark Gray)
```

- [ ] 텍스트가 명확하게 읽히는가?

- [ ] 레이블이 모든 데이터 포인트에 있는가?

## 생성 명령어 예시

```bash
# 기본 사용법
python scripts/generate_visual.py "프롬프트" -o figures/파일명.png

# 문서 유형 지정
python scripts/generate_visual.py "프롬프트" -o figures/파일명.png --doc-type report

# 품질 임계값 조정
python scripts/generate_visual.py "프롬프트" -o figures/파일명.png --threshold 8.0

# 최대 반복 횟수 지정
python scripts/generate_visual.py "프롬프트" -o figures/파일명.png --max-iterations 5
```

### 문서 유형별 품질 임계값

| 문서 유형 | 임계값 | 설명 |
|----------|--------|------|
| journal | 8.5 | 학술 저널 수준 |
| report | 7.5 | 비즈니스 리포트 |
| presentation | 6.5 | 프레젠테이션 |
