# Persona Framework (Fashion Edition)

Wacky Willy의 팬덤이 될 가상의 인물(Persona)을 패션 라이프스타일 중심으로 구체화하는 도구입니다.

## Persona Components (Fashion Context)

### 1. Fashion ID (Identity)

```markdown
## Fashion ID
| 항목 | 내용 |
|------|------|
| **이름** | [가상의 한국/영어 닉네임] e.g., Kiki (키키) |
| **나이** | 22세 |
| **직업** | 그래픽 디자이너 / 대학생 |
| **거주지** | 서울 마포구 (홍대/연남동) |
| **Spending** | 월 의류 지출 30~50만원 |
| **Style** | Kitsch, Street, Layered |
```

### 2. OOTD (Outfit Of The Day) Profile

*   **Top**: 오버사이즈 그래픽 티셔츠, 빈티지 바시티 재킷
*   **Bottom**: 와이드 카고 팬츠, 로우라이즈 데님
*   **Shoes**: 청키한 스니커즈 (커스텀)
*   **Acc**: 키링 주렁주렁 백팩, 비니

### 3. Digital Habitat (디지털 서식지)

*   **Instagram**: 힙한 카페 투어, 거울 셀카 (@ootd_kiki)
*   **TikTok**: 댄스 챌린지보다는 'Unboxing', 'Grwm' 시청
*   **Shopping App**: 무신사, KREAM(리셀), 번개장터(빈티지)
