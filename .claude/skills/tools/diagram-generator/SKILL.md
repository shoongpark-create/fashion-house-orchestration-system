# Fashion Diagram Generator

패션 비즈니스 전략 수립을 위한 시각화 도구입니다.

## Diagram Types

### 1. Positioning Map
*   **Axes**: Price (Low-High) vs Fashionability (Basic-Trendy)
*   **Purpose**: 우리 브랜드의 위치 확인 (Niche Market 발굴)

### 2. Line Sheet Structure
*   **Format**: Tree Map
*   **Content**: Category breakdown (Outer, Top, Bottom, Acc)

### 3. Collection Mood Board
*   **Format**: Grid Layout
*   **Content**: Color Palette, Texture Swatches, Inspiration Images

## Usage
이 스킬은 `chief-merchandiser`가 전략 수립 시 시각 자료를 생성할 때 사용합니다.
